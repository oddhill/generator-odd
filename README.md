# generator-odd
Yeoman generator for [Odddrupal](https://github.com/oddhill/odddrupal) and [Oddbaby](https://github.com/oddhill/oddbaby)

## Install
```
npm install -g generator-odd
```

## How to use

```
yo odd:drupal
yo odd:baby
```
Files and folders will be put in the cwd.

## Tests
Using mocha for tests.
```
npm test
```
